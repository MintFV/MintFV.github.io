# Benötigte Bilder

Für die vollständige Funktion der Website werden noch folgende Bilder benötigt:

## Header-Bild für die Startseite
- `assets/images/header-bg.jpg` - Hintergrundbild für den Header auf der Startseite

## Feature-Row Thumbnails (Startseite)
Diese Bilder werden auf der Startseite in der Feature-Row angezeigt:
- `assets/images/veranstaltungen-thumb.jpg` - Bild für "Veranstaltungen"
- `assets/images/mitgliedschaft-thumb.jpg` - Bild für "Mitgliedschaft"
- `assets/images/instagram-thumb.jpg` - Bild für "Social Media"

## Empfohlene Bildgrößen
- Header-Bild: ca. 1920x1080px (16:9 Format)
- Feature-Row Thumbnails: ca. 600x400px (3:2 Format)

## Alternative
Wenn Sie keine eigenen Bilder haben, können Sie:
1. Das Header-Bild weglassen (entfernen Sie `overlay_image` aus `index.md`)
2. Die Thumbnails weglassen (entfernen Sie `image_path` aus der `feature_row` in `index.md`)
3. Platzhalter-Bilder von [Unsplash](https://unsplash.com) oder [Pexels](https://pexels.com) verwenden
